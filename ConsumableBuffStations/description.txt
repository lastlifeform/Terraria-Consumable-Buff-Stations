This is a small QOL mod that allows you to shimmer buff stations into “permanent booster” consumables, letting you have their buffs at all times.

--

The following transmutations exist:

Ammo Box => Ballistic Popcorn Bucket

Bewitching Table => Witch Kebab

Crystal Ball => Clairvoyant Gumball

Sharpening Station => Peppermint Grindstone 

War Table => Strategic Smorgasbord


If Thorium is installed:
Altar => Golden Wafer
Conductor’s Stand => Composers Candy
Ninja Rack => Ninja Ribs
